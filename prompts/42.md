# 비주얼모드 Level 1 툴팁 비활성화 요청

## 현재 상태

모든 메뉴 레벨에 `desc` 필드를 추가하고, hover 시 설명 툴팁이 표시되도록 구현되었습니다.

다만 비주얼모드에서는 Level 1 메뉴가 실제 편집 대상이라기보다는 상단 탭/카테고리 전환 용도로 사용됩니다.

따라서 비주얼모드에서는 Level 1 메뉴에 hover했을 때 설명 툴팁이 표시되지 않아도 됩니다.

---

## 수정 요청

비주얼모드에서만 Level 1 메뉴의 설명 툴팁을 비활성화해 주세요.

```txt
비주얼모드 Level 1 hover → 툴팁 표시 안 함
````

---

## 유지할 동작

아래 동작은 그대로 유지해 주세요.

```txt
리스트모드 Level 1 툴팁 표시 유지
리스트모드 Level 2 툴팁 표시 유지
리스트모드 Level 3 툴팁 표시 유지

비주얼모드 Level 2 툴팁 표시 유지
비주얼모드 Level 3 툴팁 표시 유지

미리보기 화면 툴팁 기존 동작 유지
```

즉, 이번 수정은 **비주얼모드의 Level 1 툴팁만 끄는 작업**입니다.

---

## 적용 기준

비주얼모드에서 메뉴 아이템의 level이 1이면 `desc` 값이 있어도 툴팁을 렌더링하지 않도록 처리해 주세요.

예시:

```js
const shouldShowDescTooltip =
  mode !== "visual" || menu.level !== 1;
```

또는 컴포넌트 내부에서:

```jsx
<MenuDescTooltip
  desc={menu.desc}
  disabled={isVisualMode && menu.level === 1}
>
  {children}
</MenuDescTooltip>
```

| 화면    | Level 1  | Level 2  | Level 3  |
| ----- | -------- | -------- | -------- |
| 리스트모드 | 툴팁 표시    | 툴팁 표시    | 툴팁 표시    |
| 비주얼모드 | 표시 안 함   | 툴팁 표시    | 툴팁 표시    |
| 미리보기  | 기존 동작 유지 | 기존 동작 유지 | 기존 동작 유지 |
