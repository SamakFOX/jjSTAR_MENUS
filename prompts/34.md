# Google Font 제거 및 Pretendard WOFF2 로컬 폰트 적용 요청

## 작업 목표

현재 프로젝트에서 Google Font 사용을 제거하고, 프로젝트 내부에 저장된 Pretendard WOFF2 폰트를 전역 폰트로 적용해 주세요.

현재 폰트 파일 위치는 다음과 같습니다.

```txt
src/resources/PretendardVariable.woff2
````

이번 작업은 **폰트 적용만** 대상으로 합니다.

---

## 수정 범위

수정 대상 파일은 주로 아래 두 파일입니다.

```txt
src/app/layout.js

```


## 1. Google Font 관련 코드 제거

`layout.js`, `globals.css`, 또는 기타 파일에서 Google Font 관련 코드를 제거해 주세요.

제거 대상 예시:

```js
import { Inter } from "next/font/google";
import { Noto_Sans_KR } from "next/font/google";
import { Noto_Sans } from "next/font/google";
```

아래처럼 Google Font 인스턴스를 만드는 코드도 제거해 주세요.

```js
const inter = Inter({
  subsets: ["latin"],
});

const notoSansKr = Noto_Sans_KR({
  subsets: ["latin"],
});
```

`html` 또는 `body`에 적용되어 있던 기존 Google Font className도 제거해 주세요.

```jsx
<html lang="ko" className={inter.className}>
```

```jsx
<body className={notoSansKr.className}>
```

---

## 2. Google Font 외부 요청 제거

아래와 같은 외부 Google Font 요청 코드가 있다면 제거해 주세요.

```html
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com">
<link href="https://fonts.googleapis.com/..." rel="stylesheet">
```

```css
@import url("https://fonts.googleapis.com/css2...");
```

최종적으로 Google Fonts로 나가는 외부 요청이 없어야 합니다.

---

## 3. next/font/local로 Pretendard 적용

Next.js 프로젝트이므로 `next/font/local`을 사용해 로컬 폰트를 적용해 주세요.

`src/app/layout.js` 기준으로 폰트 파일 경로는 다음입니다.

```txt
../resources/PretendardVariable.woff2
```

`src/app/layout.js` 예시:

```jsx
import localFont from "next/font/local";
import "./globals.css";

const pretendard = localFont({
  src: "../resources/PretendardVariable.woff2",
  variable: "--font-pretendard",
  weight: "45 920",
  display: "swap",
});

export const metadata = {
  title: "JJSTAR 메뉴 배치 테스트",
  description: "JJSTAR 메뉴 배치 사용자 테스트",
};

export default function RootLayout({ children }) {
  return (
    <html lang="ko" className={pretendard.variable}>
      <body>{children}</body>
    </html>
  );
}
```

기존 `metadata` 내용이 이미 있다면 그대로 유지하고,
폰트 관련 부분만 위 방식으로 반영해 주세요.

## 4. globals.css에서 전역 폰트 적용

`src/app/globals.css`에서 `html`, `body`에 Pretendard를 전역 적용해 주세요.

```css
html,
body {
  font-family: var(--font-pretendard), system-ui, -apple-system,
    BlinkMacSystemFont, "Segoe UI", sans-serif;
}
```

기존에 `body` 또는 `*`에 다른 `font-family`가 지정되어 있다면, Pretendard가 우선 적용되도록 정리해 주세요.

예시:

```css
body {
  font-family: var(--font-pretendard), system-ui, sans-serif;
}
```

## 5. Tailwind 사용 시 추가 확인

프로젝트에서 `font-sans` 클래스를 많이 사용하고 있다면,
전역 CSS만으로도 대부분 적용될 수 있습니다.

다만 Tailwind 설정에서 `font-sans`가 다른 폰트로 지정되어 있다면,
`tailwind.config.js` 또는 `tailwind.config.mjs`에서 아래처럼 설정해 주세요.

```js
theme: {
  extend: {
    fontFamily: {
      sans: ["var(--font-pretendard)", "system-ui", "sans-serif"],
    },
  },
},
```

Tailwind 설정 파일이 없거나 별도 설정이 필요 없다면 이 단계는 생략해도 됩니다.

## 6. 직접 @font-face 방식은 이번 작업에서 사용하지 않기

아래 방식도 가능하지만, 이번 프로젝트에서는 `next/font/local` 방식을 우선 사용해 주세요.

```css
@font-face {
  font-family: "Pretendard";
  src: url("../resources/PretendardVariable.woff2") format("woff2");
  font-weight: 45 920;
  font-style: normal;
  font-display: swap;
}
```

이번 작업에서는 위 방식보다 `next/font/local` 방식으로 적용해 주세요.
