# UI 문구 한글 원상복귀 요청

## 현재 상태

인증코드 기반 로그인 시스템과 Supabase 저장 기능은 정상적으로 구현되었습니다.

현재 확인된 정상 동작:

```txt
인증코드 로그인 정상 동작
/api/verify-code 정상 동작
/api/submit 정상 동작
Supabase submissions 저장 정상 동작
auth_codes submit_count 증가 정상 동작
```

하지만 구현 과정에서 기존 UI 문구가 영어로 변경되었습니다.

이번 작업은 **기능 수정이 아니라 UI 문구 원상복귀 작업**입니다.

---

# 작업 목표

영어로 바뀐 UI 문구를 기존 한글 UI 문구로 되돌려주세요.

중요:

```txt
인증 로직 수정 금지
API Route 수정 금지
Supabase 저장 로직 수정 금지
drag/drop 로직 수정 금지
List Mode 로직 수정 금지
Visual Editor Mode 로직 수정 금지
menuTree 구조 수정 금지
```

이번 작업은 **텍스트/라벨/버튼 문구만 한글로 복구**하는 작업입니다.

---

# 유지해야 하는 기능

아래 기능은 현재 정상 동작하므로 절대 건드리지 마세요.

```txt
인증코드 로그인
sessionStorage 로그인 유지
/api/verify-code
/api/submit

auth_codes submit_count 증가

Visual Editor Mode

Level 1/2/3 제한

```

# 로그인 화면 문구

로그인 화면은 아래처럼 한글로 표시해주세요.

## 제목

```txt
JJSTAR 메뉴 배치 테스트
```

## 안내 문구

```txt
인증코드를 입력하면 메뉴 배치 테스트에 참여할 수 있습니다.
```

또는 기존 UI 분위기에 맞게:

```txt
발급받은 인증코드를 입력해주세요.
```

## 입력 placeholder

| 입력 항목 | placeholder |
|---|---|
| 인증코드 | 인증코드를 입력해주세요 |
| 이름/닉네임 | 이름 또는 닉네임을 입력해주세요 |

## 버튼

```txt
입장하기
```

## 오류 문구

| 상황 | 문구 |
|---|---|
| 인증코드 미입력 | 인증코드를 입력해주세요. |
| 이름 미입력 | 이름 또는 닉네임을 입력해주세요. |
| 인증 실패 | 유효하지 않은 인증코드입니다. |
| 비활성 코드 | 비활성화된 인증코드입니다. |
| 만료 코드 | 만료된 인증코드입니다. |
| 제출 횟수 초과 | 제출 가능 횟수를 초과한 인증코드입니다. |
| 서버 오류 | 인증 처리 중 오류가 발생했습니다. |

# 리스트 편집 모드 문구

`MenuBoard` 관련 기존 한글 문구를 유지해주세요.

예시:

```txt
Level 1 추가
Level 1 순서변경
순서변경 완료
취소
되돌리기
Level 1만 보기
Level 2까지 보기
전체 펼치기
대분류(Level 1) 순서 변경 중
```

기존 alert/confirm도 한글로 유지합니다.

예시:

```txt
Level 1 메뉴는 다른 메뉴의 하위로 들어갈 수 없습니다.
Level 3 메뉴는 Level 2 하위에만 배치할 수 있습니다.
Level 3 메뉴는 하위 메뉴를 가질 수 없습니다.
작업이 취소되었습니다.
메뉴 이름이 변경되었습니다.
새 대분류가 추가되었습니다.
새 하위 메뉴가 추가되었습니다.
```

# Drag & Drop 관련 표시 문구

드래그 중 표시되는 문구도 한글로 바꿔주세요.

| 현재 영어 문구 예시 | 한글 문구 |
|---|---|
| MOVE HERE | 여기로 이동 |
| DROP SUB-ITEM | 하위로 이동 |
| Invalid | 불가 |
| Not allowed | 이동 불가 |

단, 내부 코드의 `dropMode`, `reorder`, `move-to-parent` 같은 값은 로직용이므로 변경하지 마세요.

# 최종 요약

인증코드 로그인과 Supabase 제출 저장 기능은 정상 동작합니다.

이번 요청은 기능 수정이 아니라,  
구현 과정에서 영어로 바뀐 UI 문구를 기존 한글 UI로 원상복귀하는 작업입니다.

```txt
기능은 그대로 유지
영어 문구만 한글로 복구
API / Supabase / drag-drop 로직 수정 금지
```
